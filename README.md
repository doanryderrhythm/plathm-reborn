# INFORMATION
Official Plathm -Reborn- source code made from **Unity Engine**.\
**Current version:** BETA, in development.

# INSTALLATION
To check the source code and how the game works, Unity Engine must be installed from Unity Hub.\
For IDE, Visual Studio and Visual Studio Code are both recommended.

# BRANCHES
- To access the main game, go to branch main.
- To access the chart editor, go to branch editor. **WARNING:** Do not merge the branch into the main branch!
